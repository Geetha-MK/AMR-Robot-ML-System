# 🤖 Autonomous Multitasking Rescue Robot in Unmanned Zones

## 📌 Overview
The **Autonomous Multitasking Rescue Robot** is an embedded and IoT-based system designed to assist in rescue operations in hazardous environments such as fire accidents, gas leaks, and disaster zones.

The robot can:
- Detect environmental hazards (fire, gas, metal)
- Navigate autonomously
- Monitor victim health (heartbeat & temperature)
- Send real-time data wirelessly
- Assist rescue using robotic arm

---

## 🎯 Objective
- Reduce human risk in disaster environments
- Detect hazards and victims efficiently
- Monitor health conditions in real-time
- Provide intelligent decision support using Machine Learning

---

---

## ⚙️ Working Principle
- Arduino Mega acts as central controller
- Sensors detect environmental hazards
- Ultrasonic sensor enables obstacle avoidance
- Robot navigates autonomously
- Health sensors measure heartbeat & temperature
- Data is sent via Zigbee & Bluetooth
- Machine learning model predicts victim condition
- Robot assists using pick-and-place arm

---

## 🧩 Components Used

### 🔹 Hardware
- Arduino Mega 2560  
- Ultrasonic Sensor (HC-SR04)  
- Fire Sensor  
- Gas Sensor (MQ series)  
- Metal Detector  
- Heartbeat Sensor  
- Temperature Sensor (DHT11 / DS18B20)  
- DC Motors  
- Servo Motors (Robotic Arm)  
- Motor Driver (L293D H-Bridge)  
- Bluetooth Module (HC-05)  
- Zigbee Module  
- LCD Display  
- Buzzer  
- Power Supply
